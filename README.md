# Advanced CSS: Flexbox & Dark Mode Exercises

A practice webpage focused on mastering modern layout techniques

## 🎯 Learning Objectives
* Master flexible layouts using **CSS Flexbox**.
* Implement dynamic **Dark Mode** toggling using CSS Custom Properties (Variables) and JavaScript.

## 🛠️ Technologies Used
* HTML5
* Advanced CSS3 (Flexbox, Custom Properties, Media Queries)
* JavaScript (Theme switching logic)

## 🚀 How to Run
1. Clone this repository.
2. Open any `index.html` file directly in your browser.
